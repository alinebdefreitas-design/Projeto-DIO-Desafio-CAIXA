Depois pedi que o Copilot criasse um modelo de proposta para uma cliente, mãe solo, que recebe a pensao do filho e trabalha em casa com venda de personalizados para festas infantis.  O resultado foi esse: Proposta de Organização Financeira. Cliente: (Nome da cliente) Perfil: Mãe solo | Trabalho em casa | Venda de personalizados infantis
